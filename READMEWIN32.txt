Fichier README de la version Windows de TORCS

Le portage est effectué sur Windows 2000. Pas d'essai sur autres plateformes Windows pour l'instant (08/2000).

Compilation:
	A partir d'une installation 'nue' (sources uniquement), il faut:
		exécuter le fichier setup_win32.bat qui est dans la racine de votre installation TORCS
		choisissez 'AA' comme étant la configuration active (en mode debug ou ndebug, au choix)
		compilez-le

	Les versions debug ont un 'd' ajouté à la fin du nom de chaque fichier .lib et .dll .
	Si vous en avez pour 10 minutes, c'est le maximum. Aller, 15, le temps de trouver les projets...

Voilà voilà.
