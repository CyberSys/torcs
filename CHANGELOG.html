Version 1.1.0 codename Titouan:

System:

  • Windows 9x port by Christophe "Guions" Guionneau, including VC++6.0 project files.
  • Support of Windows XP and 2000 (untested on NT)
  • Use of GLUT for Win32 from Nate Robins (works on all Windows versions)

Graphic:

  • Lights, Smoke and skid marks by Guions.
  • Shadows are enhanced, with less display bug.
  • Support of png image format in .ac files.
  • Correction of png loading code (now compatible with transparent images generated by The Gimp).
  • One level of LOD based on the size of the cars on the screen.

Simulation:

  • Better stability of the cars at rest.
  • New aerodynamics model for the ground effect.
  • Time control with +/- keys during race.

Controls:

  • Support of game-pads and mouse to drive the cars in addition to the current steering wheel support.
  • Support of multiple controllers.

Tracks:

  • Trees without black outline, many thanks to Steve Baker for his valuable explanations :-)
  • Michigan Speedway is now usable, the cars don't fly away anymore.
  • Tracks sorted in categories (Oval, Road and Dirt)
  • New track format allowing kerbs and side walls
  • Track editor (work in progress)

Cars:

  • New McLaren F1, 360 modena, Lotus GT1, buggy and acura skins from Jean-Christophe "Skinner" Durieu.
  • New Porsche GT1 from Christophe "Guions" Guionneau.
  • New 360 modena from Eric "Damned" Espie.
  • New Jaguar XJ-220 derived from a model done by ViPeR and skinned by Guions.
  • 3D models enhanced (less visual defaults).

Robots:

  • Berniw(s) enhanced.
  • Inferno enhanced.
  • Inferno2 created (360 Modena).
  • Inferno3 created (Porsche GT1).
  • Inferno4 created (Jaguar XJ-220).

GUI:

  • Drivers selection enhanced, now the cars and categories are displayed for the selected robot and the keyboard can be used to navigate in the scroll lists.
  • Track selection now shows the info on the track plus a map.
  • Laps number or race distance is now configurable in an option screen.
  • Race messages are displayed during Quick Race.

Bug Corrections:

  • [ #549339 ] Crash in windows 2000
  • [ #554910 ] berniw10 crashes on Mixed 2
  • see the project on SourceForge for all the bugs.
