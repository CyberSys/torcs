Version 1.1.0:

System:

  • Windows 9x port by Christophe "Guions" Guionneau, including VC++6.0 project files.
  • Support of Windows XP (untested on 2000 and NT)
  • Use of GLUT for Win32 from Nate Robins (works on all Windows versions)

Game/Simulation:

  • Smoke and skid marks by Guions.
  • Shadows are enhanced, there are less display bug.
  • Support of png image format in .ac files.
  • Correction of png loading code (now compatible with transparent images generated by The Gimp).
  • Better stability at rest.
  • Support of Game-Pads and mouse to control the cars in addition to the current steering wheel support.

Tracks:

  • Trees without black outline, Thanks to Steve Baker :-)

Cars:

  • New McLaren F1 and Lotus GT1 skins from Jean-Christophe "Skinner" Durieu.
  • New Porsche GT1 from Christophe "Guions" Guionneau.
  • New 360 modena from Eric "Damned" Espie.
  • 3D models enhanced (less visual defaults).

Robots:

  • Inferno enhanced.
  • Inferno2 created (360 Modena).
  • Inferno3 created (Porsche GT1).

Version 1.0.0:

Robots:

  • New robots (berniw) by Bernhard Wymann from Switzerland :-)
  • New version of K1999 provided by Rémi Coulom (nascar and buggy cars)
  • New robot Inferno based on Tanhoj code with a McLaren F1 (which is not a formula one !) car

Cars:

  • New car model Alfa Romeo 155 V6 Ti DTM generously donated by Andrea Alfieri (look here to see this superb model).
  • New car model (corvette derived from a model taken on 3D Cafe).

Game/Simulation:

  • The cars are removed when out of fuel.
  • Engine now react better when down shifting, (no more high revving).
  • A little more stability added at rest, but not yet convincing.
  • New instrument panel
  • New TV director view (F11) that select automaticaly the car to view according to the action on the track.

Tracks:

  • Wheel-1, Dirt-3, Dirt-4, Dirt-6 enhanced.
